# tasks1 - Simple Task Manager Prototype

A basic command-line task manager prototype.

## Features
- Add tasks with due dates
- List all tasks
- Search tasks by title
- Store tasks in JSON file

## Running the Application
```bash
cd tasks1
python tasks.py
```

## Commands
- `add` - Add a new task
- `list` - Show all tasks
- `search` - Search for tasks
- `quit` - Exit
